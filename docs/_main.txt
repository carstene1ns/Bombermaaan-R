<h2>Introduction</h2>

Bombermaaan is a clone of the famous Bomberman and Dynablaster games. It runs under most Win32 operating systems.

You have the choice of installing Bombermaaan and running the game from the start menu or by 
unzipping the package into a folder and starting the game from there.

See \ref controls to see the keyboard layout. Bombermaaan is controlled by the keyboard.
